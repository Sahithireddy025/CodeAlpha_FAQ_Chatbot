# CodeAlpha FAQ Chatbot

## Description
This project is an AI-based FAQ Chatbot developed as part of the CodeAlpha Artificial Intelligence Internship.

The chatbot uses Natural Language Processing and cosine similarity to match user questions with predefined FAQs.

## Technologies Used
- Python
- Tkinter
- NLTK
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install nltk scikit-learn
2. Run the chatbot:
   python chatbot.py

## Features
- NLP-based question matching
- Simple chat interface
- AI-powered responses
